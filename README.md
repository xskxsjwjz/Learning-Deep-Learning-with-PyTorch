# 从零开始的深度学习！

这个项目记载了我学习深度学习的过程。如果这对你有帮助，欢迎点个小小的 star ⭐️。

本项目参考了（动手学深度学习）等优秀资源。

## 如何使用？

1. 克隆本仓库到本地：

   ```bash
   git clone https://github.com/xskxsjwjz/Learning-Deep-Learning-with-PyTorch.git
   ```

2. 安装所需的依赖包：

   ```bash
   pip install -r requirements.txt
   ```

3. 接下来就可以运行 jupyter notebook 了！
